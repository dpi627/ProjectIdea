# Project Idea Studio - Introduction Video

A modern, professional introduction video showcasing the key features of Project Idea Studio.

## Features Highlighted

- **Opening Scene**: Branding with animated logo and tagline
- **Local Data Storage**: Browser LocalStorage and file system options
- **Light/Dark Theme**: Seamless theme switching with smooth transitions
- **Service Detection**: Real-time health monitoring with status indicators
- **Model Usage Monitor**: Track AI model usage with charts and filters
- **Auto Update Check**: Automatic version detection system
- **Simple Yearly Gantt**: Project timeline visualization
- **History Diagrams**: Beautiful Chart.js analytics
- **Features Overview**: Additional capabilities showcase
- **Closing Scene**: Call-to-action with branding

## Video Specifications

- **Duration**: 30 seconds
- **Resolution**: 1920x1080 (Full HD)
- **Frame Rate**: 30 fps
- **Format**: MP4

## Development

```bash
# Start the preview studio
npm start

# Render the final video
npm run build
```
